# Electronics Price Prediction

This project builds a machine learning pipeline to predict prices of electronic products using various regression models.

## Features
- Data preprocessing using Scikit-learn pipelines
- Handling categorical and numerical features
- Model comparison:
  - Linear Regression
  - KNN
  - Decision Tree
  - Random Forest
- Hyperparameter tuning using RandomizedSearchCV

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Workflow
1. Data cleaning
2. Feature engineering
3. Preprocessing with ColumnTransformer
4. Model training
5. Hyperparameter tuning
6. Model evaluation

## Evaluation Metrics
- MAE
- RMSE
- R² score

## Best Model
Random Forest Regressor achieved the best performance.
